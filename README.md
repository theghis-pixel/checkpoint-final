
# Todo App

## Description
Application simple de gestion de tâches (Todo List) réalisée avec HTML, CSS et JavaScript.

## Fonctionnalités
- Ajouter une tâche
- Marquer une tâche comme terminée
- Supprimer une tâche
- Interface simple et responsive

## Technologies utilisées
- HTML
- CSS
- JavaScript

## Déploiement
Projet déployé avec GitHub Pages.

## Utilisation
1. Ouvrir le site
2. Ajouter une tâche
3. Cliquer sur une tâche pour la marquer comme terminée
4. Supprimer avec le bouton X
